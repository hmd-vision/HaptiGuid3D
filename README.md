# HaptiGuide3D

A vision-based assistive navigation system that combines real-time object detection, monocular depth estimation, gesture-based Human–Robot Interaction, and directional audio feedback.

## Overview

HaptiGuide3D is an assistive navigation prototype designed to support visually impaired users in indoor environments. The system uses a single RGB camera to detect obstacles, estimate relative depth, recognise hand gestures, and provide directional audio feedback.

The project combines computer vision with HRI principles, where the user is treated as an active part of the interaction loop rather than only receiving passive feedback.
